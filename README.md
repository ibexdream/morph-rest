# MÖRPH

MÖRPH is a speculative AI-powered dream programming device concept, presented as a cinematic static landing page.

The site frames MÖRPH as a featherweight wearable disc that can set a dream intent, sense sleep-stage signals, run an on-device dream-shaping loop, and return a visual replay of the night.

## Live

This project is built as a static website for GitHub Pages.

- GitHub Pages: `https://ibexdream.github.io/morph-rest/`
- Custom domain target: `https://morph.rest/`

## Concept

MÖRPH explores a fictional closed-loop dream synthesis device:

- set a dream intent before sleep
- detect the hypnagogic window
- read EEG/EOG-style sleep signals
- classify the dream state locally
- gently nudge the dream back toward the chosen intent
- save and replay a visual trace in the morning

MÖRPH is a speculative concept. Closed-loop dream synthesis does not yet exist, but every piece points to where it could.
